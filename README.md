# DPS Games Store

DPS Games Store is a static front-end project that simulates an online video game store, inspired by platforms like Steam.  
The purpose of this project is to practice **HTML semantic structure**, **CSS layout**, and **front-end best practices**.

---

## Technologies Used

- **HTML**
- **CSS**

## Notes

This is a static project and does not include interactivity, authentication, or dynamic data (yet).
Future improvements could include responsiveness enhancements or JavaScript functionality.
